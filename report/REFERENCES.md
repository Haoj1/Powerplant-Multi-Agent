## Usage Notes

- **IEEE in-text:** Citations appear as [1], [2], [3] in order of first appearance.
- **Same source multiple times:** Reuse the same number (e.g., [1] always, never [1, 5]).
- **Placement:** Inside sentence punctuation, e.g., "As shown in [1], ..." or "Prior work [2], [3] has addressed..."
- **Reference list:** Single-spaced; double-space between entries if TGS prefers.
- **TGS:** References may be single-spaced; end of text only (not per-chapter).
